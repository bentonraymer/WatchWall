## Version 1.1
**Added:**
- The application menu bar can now be moved to any side of the screen
- You can now set a gap between boxes. The size is customizable.
- You can now round the corners of boxes. The corner radius is customizable. It can also be toggled for whether it applies to the outside edge of the application or not.
- Added a refresh button to boxes to refresh the web browser
- Added an "Advanced" Dropdown in the settings menu for deeper customization options
- Added the ability to change the background color of the app

**Fixed:**
- The logo was not rendering properly in the executables both in the menu bar and settings menu.
- Fixed an issue where two-box setups were always the same size
- Fixed an issue where boxes were not remaining 16:9 and padding was being added

**Removed:**
- The ability to toggle between sending a search query to Google or YouTube in the URL/Search Bar
